## Nomenclatura de Archivos

### Productos
Formato: `{deporte}_{equipo}_{categoria}_{numero}.jpg`

Ejemplos:
- `futbol_realmadrid_camiseta_1.jpg`
- `baloncesto_fcbarcelona_sudadera_2.jpg`
- `formula1_ferrari_gorra_1.jpg`

### Banners
Formato: `banner_{descripcion}.jpg`

Ejemplos:
- `banner_promocion_verano.jpg`
- `banner_nuevos_lanzamientos.jpg`

### Iconos
Formato: `icon_{nombre}.svg` o `.png`

Ejemplos:
- `icon_futbol.svg`
- `icon_baloncesto.svg`
- `icon_formula1.svg`

## Formatos Recomendados

- **Productos**: JPG/PNG, 800x800px, fondo blanco o transparente
- **Banners**: JPG, 1920x600px
- **Iconos**: SVG (preferido) o PNG 256x256px
